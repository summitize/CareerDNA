Premium, student-friendly SaaS web app built with Next.js 15, TypeScript and Tailwind CSS.

## Tech Stack

- Next.js 15 + React
- Tailwind CSS
- Framer Motion
- ShadCN-style reusable UI components
- React Hook Form + Zod
- React Query
- Chart.js
- Supabase SDK
- Prisma ORM
- jsPDF export

## Run locally

```bash
npm install
npm run dev
```

## Key Routes

- `/` Landing page
- `/student` Student dashboard
- `/assessment` Assessment engine (autosave + timer)
- `/report` Report + PDF export
- `/mentor` AI mentor mock UI
- `/counsellor` Counsellor dashboard
- `/school-admin` School dashboard
- `/admin` Super admin panel
- `/careers` Career database
